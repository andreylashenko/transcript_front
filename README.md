# Телефонный справочник

Перед разверткой проекта убедитесь, что у вас установлены nodejs и npm.

Развертка проекта: в корне проекта выполнить следующие команды

``` bash
# install vue/cli
> npm install -g @vue/cli
# install dependencies
> npm install
# build project
> npm run build
# start project
> npm run dev
```
